Musicote Android App

This App is for listen the music (Melchor629's webapp) in your Mobile Device. This App is still in beta. If you want to help, don't doubt and do Pull Requests :D

What is this app?

This app is for listen music allocated in a computer with the melchor629's music webapp (only Linux & Mac OS X). It's in beta, but you can test it for help the developer.

How to use the API?

If you prefer to help to the developer you must know how to use the API. The API is allocated in http://reinoslokos.no-ip.org/cgi-bin/archivos.py, and returns a JSON file with all the songs and his data. It's very easy to use :D
